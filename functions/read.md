# Read Contract Functions

On Etherscan, this is the order of functions that should return variables under the "[Read Contract](https://info.etherscan.com/how-to-use-read-or-write-contract-features-on-etherscan/)" section:

1) name
2) symbol
3) website
4) exchange
5) totalSupply
6) buyTax
7) sellTax
8) owner
9) communityWallet
10) marketingWallet
11) developerWallet
12) balanceOfAddress
13) maxWallet
14) maxTransaction
15) tradingActive
16) swapEnabled
17) swapTokenAmount
18) decimals
19) uniswapV2Pair
20) uniswapV2Router
21) deadAddress
